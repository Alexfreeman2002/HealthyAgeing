# Flask Healthy Aging Website

## Overview
_**Healthy Ageing Plans**_ Website is designed to provide information, resources, and tools for promoting healthy aging. It includes features such as: 
- Semi-personalized nutrition plans 
- Semi-personalized fitness plans 
- Daily calorie intake calculator (with guidelines for calorie deficit and surplus)
- Fitness advice 
- Nutrition advice 
- Updated news feed 
- Disease searching function

## Useage
https://healthyageing.azurewebsites.net/

#### Alternatively 
Clone the repository on your local machine: git clone https://github.com/Alexfreeman2002/HealthyAgeing.git
<br> Install the dependencies using pip: pip install -r requirements.txt
<br> Run the flask application: python app.py
<br> Once running it will be available on http://127.0.0.1:5000
